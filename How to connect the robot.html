Follow the following steps to connect your robot to BUTTER MAS TERMINAL

Config Memory-card:

  1. Open Butter MAS Composer
  2. Press on the setup icon and chose "Butter Operating System Network"
  3. Insert BUD's memory card to the comuter
  4. Type : WIFI network - name + pasword

  1. Press Add Credentials
  2. Press Done
  3. Remove the memory-card and inset it to BUD

  1. Plug BUD to electrical power and wait untill the LED-LIGHT turn red.

Config BUD to Butter MAS Composer :

  1. Open CMD and type : ' ipcofig '

  2. Open Angy IP Scanner program and set the range for scanning like the ip address you can see under the line "Default Gateway" on CMD . and make sure the last number on the rage is from 0 to 255

  3. Sort the result by Hostname

  4. Look for Hostname BUD's IP address and copy the number
  5. Open Butter MAS Composer.
  6. Press "+" on deshboard , pick a name to your project and type the IP from stage 12.

  7. Press CONNECT

  8. Open the terminal TERMINAL

  9. On the bytton on down-right of the screen chose the type of your robot

  10. ###now you are connected! good luck!
